# My first Servlet

## How to create?

Windows PowerShell

```
$ mvn archetype:generate "-DgroupId=org.example" "-DartifactId=my-first-servlet" "-DarchetypeArtifactId=maven-archetype-webapp" "-DinteractiveMode=false"
```

Linux

```
$ mvn archetype:generate -DgroupId=org.example -DartifactId=my-first-servlet -DarchetypeArtifactId=maven-archetype-webapp -DinteractiveMode=false
```

## How to build?

```
$ mvn clean package
```
